# 🏆 Daily Commit Project

## 🚀 프로젝트 개요
하루에 하나의 작은 프로젝트 또는 기능을 개발하여 **GitHub에 1일 1커밋**하는 것이 목표.  
코딩 실력을 향상시키고, 꾸준한 개발 습관을 기르기 위해 진행.  

## 📌 사용 기술 및 도구

| 구분         | 도구 / 기술 |
|-------------|------------|
| **코딩 언어**    | Python |
| **코딩 에디터**  | PyCharm |
| **운영 체제** | Windows 10 |
| **버전 관리** | GitHub |
| **형상관리 전략** | Git Flow |
